## 2. X API Basic プラン使用時の実装

### 環境変数設定
```bash
# Netlifyの環境変数に設定
TWITTER_BEARER_TOKEN=your_bearer_token_here
```

### API制限対策
- **キャッシュ機能**: 1時間ごとに更新
- **優先度設定**: 重要候補者を優先取得
- **フォールバック**: API制限時はサンプルデータ使用

## 3. 段階的実装プラン

### フェーズ1: サンプルデータ（現在）
- 手動で作成したサンプルツイート
- UI/UX の完成度を高める

### フェーズ2: 限定的リアルデータ
- 主要候補者5-10名のツイート
- 1日1-2回の更新

### フェーズ3: 本格運用
- 全候補者のツイート取得
- リアルタイム更新

## 4. コスト比較

### 無料プラン
- **コスト**: $0
- **制限**: 月1,500ツイート
- **候補者数**: 3-5名程度

### Basic プラン
- **コスト**: 月$100
- **制限**: 月50,000ツイート
- **候補者数**: 50-100名程度

### 代替手段
- **コスト**: $0-20
- **制限**: なし（技術的制約のみ）
- **候補者数**: 制限なし

- X API利用規約の確認

### 高優先度

# 衆議院選挙2025 情報サイト

2025年2月実施予定の衆議院選挙に関する最新ニュースと情報を提供するWebサイトです。

## 主な機能

### 🗞️ 自動ニュース更新
- 最新の衆院選関連ニュースを自動で取得・表示
- 5分間隔での自動更新機能
- 複数の情報源からの総合的な情報収集

### 🏛️ 政党別タブ表示
- 自民党、立憲民主党、公明党、国民民主党、共産党、日本維新の会、れいわ新選組、参政党
- 各政党に関連するニュースを分類表示
- 政党別の詳細ページでより深い情報を提供

### ⭐ お気に入り保存機能
- ニュース記事のお気に入り保存
- 候補者情報のお気に入り保存
- データのエクスポート・インポート機能
- ローカルストレージによる永続化

### 🗾 選挙区マップ
- 全国の選挙区を地図上で表示
- 各選挙区の立候補者情報
- 都道府県・政党による絞り込み機能
- インタラクティブな地図操作

### 📊 議席予測表示
- 最新の世論調査に基づく議席予測
- 政党別の詳細な予測データ
- 複数シナリオでの分析
- グラフとチャートによる視覚的表示

## 技術スタック

- **フロントエンド**: Vue.js 3 + Vite
- **ルーティング**: Vue Router 4
- **地図表示**: Leaflet + OpenStreetMap
- **チャート**: Chart.js + Vue-ChartJS
- **HTTP通信**: Axios
- **スタイリング**: CSS3 (カスタムスタイル)

## セットアップ

### 必要な環境
- Node.js 16.0以上
- npm または yarn

### インストール手順

1. リポジトリのクローン
```bash
git clone <repository-url>
cd election-news-site
```

2. 依存関係のインストール
```bash
npm install
```

3. 開発サーバーの起動
```bash
npm run dev
```

4. ブラウザで http://localhost:3000 にアクセス

### ビルド
```bash
npm run build
```

## 機能詳細

### ニュース自動更新
- 5分間隔でのバックグラウンド更新
- 複数の情報源からのデータ統合
- リアルタイムでの新着ニュース通知

### 政党別表示
- 8つの主要政党に対応
- 政党ごとの専用ページ
- 関連ニュースの自動分類

### お気に入り機能
- ニュース記事と候補者の保存
- JSON/CSV形式でのデータエクスポート
- 他デバイスとのデータ同期対応

### 地図機能
- 全国289の小選挙区を表示
- 候補者情報のポップアップ表示
- 地域別・政党別の絞り込み

### 議席予測
- 統計的モデリングによる予測
- 複数シナリオでの分析
- 信頼区間付きの詳細データ

## データソース

現在はサンプルデータを使用していますが、以下のAPIとの連携を想定：

- 各種報道機関のニュースAPI
- 選挙管理委員会の公式データ
- 世論調査機関のデータ
- 地理情報システム（GIS）データ

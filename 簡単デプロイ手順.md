# 衆議院選挙2025サイト - 簡単デプロイ手順

## 方法1: Netlify Drop（最も簡単）

### 手順
1. **ビルドを実行**
   ```bash
   npm run build
   ```

2. **Netlifyにアクセス**
   - https://www.netlify.com にアクセス
   - アカウント作成（無料）

3. **ドラッグ&ドロップでデプロイ**
   - 「Want to deploy a new site without connecting to Git?」セクションを探す
   - 生成された `dist` フォルダをドラッグ&ドロップ
   - 自動的にサイトが公開される

### メリット
- Gitの知識不要
- 5分で完了
- 無料

### デメリット
- 更新時は手動でビルド&アップロード必要

## 方法2: GitHub + Netlify（推奨）

### 事前準備
1. **Gitをインストール**
   - https://git-scm.com/download/win からダウンロード
   - デフォルト設定でインストール

2. **GitHubアカウント作成**
   - https://github.com でアカウント作成
   - 新しいリポジトリ「election-site-2025」を作成

### デプロイ手順
1. **プロジェクトをGitHubにアップロード**
   ```bash
   git init
   git add .
   git commit -m "初回コミット"
   git remote add origin https://github.com/あなたのユーザー名/election-site-2025.git
   git push -u origin main
   ```

2. **Netlifyでデプロイ**
   - https://www.netlify.com でアカウント作成
   - 「New site from Git」をクリック
   - GitHubを選択してリポジトリを連携
   - ビルド設定:
     - Build command: `npm run build`
     - Publish directory: `dist`
   - 「Deploy site」をクリック

### メリット
- 自動デプロイ（コード更新時）
- バージョン管理
- 無料

## 方法3: Vercel（代替案）

### 手順
1. **Vercelにアクセス**
   - https://vercel.com
   - GitHubアカウントでログイン

2. **プロジェクトをインポート**
   - 「New Project」をクリック
   - GitHubリポジトリを選択
   - 自動的にVue.jsプロジェクトを認識
   - 「Deploy」をクリック

## 推奨: 方法1（Netlify Drop）から始める

初心者の方は方法1から始めることをお勧めします：

1. 現在のターミナルで `npm run build` を実行
2. 生成された `dist` フォルダをNetlifyにドラッグ&ドロップ
3. サイトが公開される

後で更新や機能追加が必要になったら、方法2に移行できます。
